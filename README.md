## 功能介绍
本软件采用微信小程序的原生框架，并结合微信小程序官方提供的云函数进行设计。 

本软件分为两个入口端：用户端和共享端，两个端口分别有以下几方面的功能：

### 用户端小程序（userport）：
1. 发布订单
2. 查看发布订单
3. 查看待接单信息
4. 查看我的评价 
### 共享端小程序（sharingport）：
1. 接单
2. 查看接单排行榜
3. 查看新手指南
4. 与客服进行对话（智能客服以及人工客服）
5. 查看已接单信息
6. 查看我的评价


## 代码下载即运行方法
1. git clone 本项目
2. 进入该项目文件夹
3. 导入小程序开发者工具（选择云函数开发），两个文件夹要分开导入
4. 在对应的文件夹下配置自己的appid和云数据库对应
5. 等待项目加载完成，注意要联网
